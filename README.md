# Lena Baumann — Academic Site

GitHub Pages + Jekyll (Minimal Mistakes theme)

## Setup
1. Rename repo to `USERNAME.github.io`
2. Push this folder to main branch
3. In repo Settings → Pages → Source: Deploy from branch (main, /root)

## Edit content
- `_config.yml` — site title, USERNAME, research line
- `index.md` — home
- `_pages/research.md` — publications
- `_posts/` — blog posts (music theory)
- Math works via MathJax: use `$...$` or `$$...$$`

## Local preview (optional)
```
bundle install
bundle exec jekyll serve
```
